## 📋 Sobre o Projeto

Landing page desenvolvida para **Dra. Ana Clara Oliveira** (persona fictícia), psicóloga especializada em Terapia Cognitivo-Comportamental. O projeto foi criado com foco em **alta conversão**, **experiência do usuário** e **performance**.

### ✨ Características Principais

- 🎨 **Design Moderno** - Interface limpa e profissional com paleta de cores suaves
- ⚡ **Animações Avançadas** - Efeitos grandiosos usando AOS Library e CSS animations
- 📱 **100% Responsivo** - Otimizado para todos os dispositivos (mobile, tablet, desktop)
- ♿ **Acessível** - Seguindo padrões WCAG 2.1 nível AA
- 💬 **WhatsApp Integrado** - Formulário completo que gera mensagem estruturada
- 🚀 **Performance Otimizada** - Lazy loading, minificação e otimização de assets
- 🎯 **SEO Friendly** - Meta tags, HTML semântico e estrutura otimizada

## 🚀 Como Usar

### Online (Recomendado)
Acesse diretamente: **[https://tofariasti.github.io/landing-page-psicologa/](https://tofariasti.github.io/landing-page-psicologa/)**

### Localmente

1. **Clone o repositório**
```bash
git clone https://github.com/tofariasti/landing-page-psicologa.git
cd landing-page-psicologa
```

2. **Abra em um servidor local**
```bash
# Opção 1: Python
python3 -m http.server 8000

# Opção 2: Node.js (com http-server)
npx http-server

# Opção 3: PHP
php -S localhost:8000
```

3. **Acesse no navegador**
```
http://localhost:8000
```

## 🔧 Personalização

### Alterar Cores
Edite as variáveis CSS em `site/css/style.css`:
```css
:root {
  --primary-color: #9b7ed6;      /* Cor primária */
  --secondary-color: #e8b4d9;    /* Cor secundária */
  --accent-color: #ffd6e8;       /* Cor de destaque */
}
```

### Alterar Número do WhatsApp
Em `site/js/main.js`, localize:
```javascript
const phoneNumber = '5511987654321'; // Substitua pelo seu número
```

### Alterar Conteúdo
Edite `site/index.html` para modificar textos, imagens e seções.

### Adicionar Imagens Próprias
Substitua as URLs do Unsplash pelas suas imagens:
```html
<img src="sua-imagem.jpg" alt="Descrição">
```
